# Magnetic Vision

Magnetic Vision è un progetto di creative coding che trasforma lo smartphone in un organo percettivo artificiale ispirato alla magnetoricezione degli uccelli migratori.

L'app usa la fotocamera in tempo reale come base visiva e sovrappone un layer generativo composto da campo fluido, pulsazioni, particelle e interferenze. La bussola orienta il campo, il giroscopio altera la stabilità della percezione e il GPS registra la posizione geografica.

La visualizzazione non vuole riprodurre esattamente ciò che vede un uccello, ma creare una percezione artistica e scientificamente informata del campo magnetico terrestre.

## Input

- Camera dello smartphone
- Bussola / magnetometro
- Giroscopio
- GPS

## Output

- Overlay generativo in tempo reale
- Linee fluide orientate dal Nord magnetico
- Pulsazioni legate all'intensità del segnale
- Particelle e interferenze ambientali

## Avvio

Apri `index.html` con Live Server in VSCode. Su smartphone è consigliato usare HTTPS o un server locale accessibile dal dispositivo.
